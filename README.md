# Alumni_Map

An interactive map to show where JF lab alumni are and what they are doing 

GitHub pages is used to host the .html file so it can be embedded into the jflab.ca squarespace site. 

To update:
1) download the jupyter notebook and install necessary packages (mostly uses Folium).
2) Add or remove lines from markers. Include their name, latituite, longitude, job title, and workplace
3) run the code to generate an updated "Alumni_map.html" (don't change the name or it will break the link on jflab.ca)
4) commit the new code and "Alumni_map.html" file. The map on jflab.ca should update automatically in a few minutes (progress can be seen on the Actions tab)
